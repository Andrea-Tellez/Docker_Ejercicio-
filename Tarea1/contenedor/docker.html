Docker

Docker es una plataforma de software de código abierto para crear, implementar y administrar contenedores de aplicaciones virtualizados en un sistema operativo (SO) común, con un ecosistema de herramientas aliadas. La tecnología de contenedores Docker debutó en 2013; Docker Inc. se formó para respaldar una edición comercial de software de gestión de contenedores y ser el patrocinador principal de una versión de código abierto.

Docker se creó para funcionar en la plataforma Linux, pero se ha ampliado para ofrecer un mayor soporte para sistemas operativos que no son Linux, incluidos Microsoft Windows y Apple OS X. Hay disponibles versiones de Docker para Amazon Web Services (AWS) y Microsoft Azure.
